QuickBus PromoCode API Java 

INTRODUCTION
------------

Thank you for downloading the QuickBus API and Sample TestPro project.  
This download contains code designed to help you begin using PromoCode APIs.


REQUIREMENTS
------------

JDK 1.8 
Eclipse Ide

CONTENTS
--------

This sample zip contains:
#1: QuickBus PromoCode Api

    /lib - contains all dependent jar files to run the sample
    /src - code for PromoCode API
    /.classpath & .project - project files for eclipse
#2: TestPro

    /lib - contains all dependent jar files to run the sample
    /src - example code to execute the PromoCode APIs
    /.classpath & .project - project files for eclipse
#3: test.sql
	- mysql database code
#4: ReadMe file

    /readme.txt - this file

RUNNING THE EXAMPLE
-------------------

1. Unzip the files contained in the QuickBus Promo.zip file to a folder on you hard drive.  
2. Install xampp and run test.sql to import database
3. Run Application in TestPro in command window i.e. cd to your jre/bin folder and then to
   either javac or javaw.exe and run the main class here.
4. Can also run in eclipse ide

DEBUGGING THE EXAMPLE IN ECLIPSE IDE
---------------------
1. Unzip the project with Api
2. Load the project into Eclipse.  
a) Copy the unzipped folder into your Eclipse workspace
b) In Eclipse, open the TestPro project in workspace
c) Clean and build the test project and click on Run.
